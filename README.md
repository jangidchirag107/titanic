# 🚢 Titanic Data Analysis Project

A complete **Exploratory Data Analysis (EDA)** project on the famous **Titanic dataset**.
This project uncovers hidden patterns, relationships, and insights about passenger survival using Python data analysis and visualization techniques.

---

## 📌 Project Overview

The Titanic dataset is one of the most popular datasets for learning data analysis and machine learning.
In this project, we analyze passenger data to understand:

* Who survived and who didn’t
* What factors affected survival
* Trends in age, class, gender, and fare

---

## 📂 Dataset

* `titanic_dataset.csv` → Raw dataset
* `cleaned_titanic.csv` → Preprocessed dataset

### Features Included:

* PassengerId
* Name
* Sex
* Age
* Pclass (Ticket Class)
* Fare
* Embarked
* Survival (Target Variable)

## 🔍 Data Analysis Steps

### 1. Data Loading

* Imported dataset using Pandas
* Checked structure and data types

### 2. Data Cleaning

* Handled missing values
* Removed unnecessary columns
* Converted categorical data

### 3. Exploratory Data Analysis (EDA)

* Univariate Analysis
* Bivariate Analysis
* Correlation Analysis

---

## 📊 Visualizations

The project includes multiple visualizations such as:

* 📈 Count Plots (Survival count, Gender distribution)
* 📊 Bar Charts (Survival by class, gender)
* 📉 Histograms (Age distribution)
* 📦 Box Plots (Outliers in fare and age)
* 🔥 Heatmap (Correlation matrix)
* 🎯 Pairplot (Feature relationships)

---

## 💡 Key Insights

* 👩 Females had a higher survival rate than males
* 🎟️ Passengers in **1st class** had better survival chances
* 👶 Younger passengers had slightly higher survival rates
* 💰 Higher fare passengers were more likely to survive
* 🚢 Port of embarkation influenced survival

## ▶️ How to Run

1. Clone the repository

```bash

```

2. Navigate to the folder

```bash
cd titanic-eda
```

3. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

4. Run Jupyter Notebook

```bash
jupyter notebook
```
